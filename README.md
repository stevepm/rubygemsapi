# RubyGemsApi

Easy access to the RubyGems API through a convenient ruby interface

## Installation

Add this line to your application's Gemfile:

    gem 'RubyGemsApi'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install RubyGemsApi
